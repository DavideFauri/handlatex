What is handLaTex?

handLatex makes it easy to inject automated, pseudo-random variations in LaTeX documents' output reproducing the typical quirks of a left to right, top to bottom handwriting. It has two components:

  1. a LaTeX package, hand, that can be included in any valid LaTeX document.

  2. a specialized command-line frontend, handlatex, able to process LaTeX documents using the hand package.

Requirements

  • A working TeX distribution, supporting LaTeX 2e (originally developped on TeX Live)

  • Python 2.5.x

Installation

  • The hand package (composed of the files hand.dtx, hand.ins and hand.def) should be installed as any other LaTeX package, following your local TeX distribution's documentation.

  • handlatex should be installed as any other Python module, using the provided distutils script (setup.py). Typical invokation will be, with proper privileges: 

python setup.py install

Usage

The hand package comes fully documented: you can usually get a local version of the manual by invoking your favorite LaTeX driver on hand.dtx. The lasted version can be found pre-compiled here, in PDF.

Once you have a valid LaTeX document using hand, you can compile it by invoking handlatex on it. For the sake of completeness, below is reproduced its manual page (handlatex.1).

handLaTeX Command Line Frontend Manual Page
HANDLATEX(1)                     User Commands                    HANDLATEX(1)



NAME
       handlatex - manual page for handlatex 1.0.0

SYNOPSIS
       handlatex [options] input_file

DESCRIPTION
       Process the LaTeX input_file with the handwriting driver.

OPTIONS
       --version
              show program's version number and exit

       -h, --help
              show this help message and exit

       --lowwordangle=LOWWORDANGLE
              force  lowwordangle option (see hand package doc; default value:
              -2)

       --maxparscale=MAXPARSCALE
              force maxparscale option (see hand package doc;  default  value:
              1.25)

       --minparscale=MINPARSCALE
              force  minparscale  option (see hand package doc; default value:
              0.8)

       --encoding=ENCODING
              force encoding option (see  hand  package  doc;  default  value:
              utf-8)

       --highwordangle=HIGHWORDANGLE
              force highwordangle option (see hand package doc; default value:
              2)

       --driver=DRIVER
              force driver option (see hand package doc; default value: latex)

       --minparangle=MINPARANGLE
              force  minparangle  option (see hand package doc; default value:
              -2)

       --freqword=FREQWORD
              force freqword option (see hand package doc; default value: 0.4)

       --maxparangle=MAXPARANGLE
              force  maxparangle  option (see hand package doc; default value:
              2)
